# Skill: Build Feature

## Objective
Implement the feature as defined in the approved spec. Verify it works before reporting completion.

## Steps
1. Read Technical_Specification.md first — do not proceed without it
2. Build the feature
3. Open the browser and take a screenshot of it working
4. Report: what you built, any trade-offs you made, any spec deviations (and why)
