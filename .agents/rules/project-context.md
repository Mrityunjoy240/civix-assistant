# Project Context

## Tech Stack
- Frontend: Next.js 15+ (App Router) + Tailwind CSS + React
- Backend: API Routes (Next.js)
- Database: TBD based on challenge (Firebase/Supabase preferred)
- Deployment: Vercel / Google Cloud Run

## Architecture Principles
- Keep components small and single-purpose
- Document every function with a one-line comment
- All environment variables in .env — never hardcode secrets
- Prioritize Server Components and 'Server-First' architecture
- AI-ready architecture with scalable components

## Output Standards
- TypeScript for all frontend and backend code
- Type all API responses
- Every page must be mobile-responsive
